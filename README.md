# 🔍 Regex Lab

Live regex tester with real-time highlighting, group inspection, replace mode, and shareable links.

## Features
- **Live matching** — highlights all matches as you type
- **Capture group display** — see $1, $2, $3... for each match
- **Replace mode** — test `$1`, `$2` replacements live
- **8 built-in presets** — email, URL, IPv4, phone, date, hex, UUID, words
- **Save your patterns** — local storage, quick recall
- **Share via URL hash** — copy link includes pattern, flags, and test string
- **Cheat sheet** always visible
- **Error display** for invalid regex
- Zero dependencies

## Run
Open `index.html`.
